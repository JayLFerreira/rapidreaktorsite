Capabilities

Features

⚡
Auto-Extract IPs from Syslogs

Smart pattern matching identifies threat IPs from any syslog source. Whether it's behind a NAT, a proxy, or a multi-hop scenario, Rapid Reaktor finds the true source IP automatically.

📋
Dynamic EDL Management

Turn your server into a live defense feed. Rapid Reaktor hosts blocklists that your firewall pulls from in real-time: EDLs for Palo Alto, Threat Feeds for Fortinet, or Security Intelligence for Cisco.

🐳
Docker Deployable

Spin up Rapid Reaktor in seconds. No complex dependencies or messy environments. Just pull the container, set your ports, and start blocking threats immediately.

🎯
Granular Trigger Logic

Go beyond simple blocking. Define custom regex patterns and logic thresholds to ensure you only block high-confidence threats, eliminating false positives and keeping legitimate traffic flowing.

⏱️
Smart TTL Management

Prevent blocklist bloat. Set custom Time-to-Live (TTL) values to automatically expire old IPs. Keep your firewall tables clean and your performance high without lifting a finger.

📊
Audit & Compliance Logging

Maintain full visibility. Track every extracted IP and automated block with detailed activity logs. Search history, export reports, and keep a perfect audit trail for compliance.
